Hate Speech Detection using Machine Learning

Project Report

Abstract

This project focuses on detecting hate speech in text using Natural Language Processing (NLP) and Machine Learning. A Support Vector Machine (LinearSVC) model trained on a balanced dataset is used with TF-IDF vectorization to classify text as either Hate Speech (1) or Non-Hate Speech (0).

Methodology

  • Data preprocessing (cleaning, tokenization, balancing dataset)
  • Feature extraction using TF-IDF Vectorization
  • Model training with Support Vector Classifier (Linear SVC)
  • Evaluation using precision, recall, F1-score, and ROC-AUC
  • Deployment using a Flask API with a simple web interface

Conclusion

The trained model achieved high accuracy and good recall for detecting offensive language. The Flask-based deployment ensures real-time predictions with a user-friendly interface.
